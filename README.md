## About

NGO Management Platform is a Django-based web application designed to support the operational and engagement needs of non-governmental organizations.

The system enables NGOs to manage member registrations, charity initiatives, donation workflows, and organizational communication through a centralized platform. It also includes a media gallery system for sharing NGO activities and events.

The platform was originally developed for the NGO **Unique Group of Mankind** and was actively used in production to manage member participation, charity alerts, and donation tracking.

This repository contains a **sanitized development version** of the platform intended for demonstration and portfolio purposes.
